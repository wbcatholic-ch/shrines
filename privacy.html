1. 기본 원칙

가톨릭길동무는 매일미사, 성가, 성경, 주요 기도문, 성당·성지·피정의집 찾기, 성지 순례 기록, 경로검색, 가톨릭 웹사이트, 관구·교구 정보, 나의 신앙생활, 문의·건의 기능을 제공하는 가톨릭 신앙생활 안내 앱입니다.

가톨릭길동무는 앱 제공에 필요한 범위에서만 정보를 이용하며, 사용자의 개인정보를 불필요하게 요구하지 않습니다.

문의 기능을 이용할 때 이름, 전화번호, 주소, 주민등록번호, 건강정보, 금융정보 등 민감하거나 불필요한 개인정보를 입력하지 않도록 주의해 주세요.

2. Google Play 데이터 보안 요약

가톨릭길동무는 회원가입, 로그인, 결제, 광고 맞춤화를 제공하지 않습니다.

앱은 연락처, 통화기록, 문자메시지, 사진·동영상, 마이크, 카메라, 금융정보, 건강정보를 요청하거나 수집하지 않습니다.

위치 정보는 사용자가 내 주변 찾기, 현재 위치, 경로검색, 길찾기, 성지 순례 GPS 자동 등록 기능을 직접 실행할 때만 사용되며, 가까운 장소 표시, 길찾기 계산, 방문 가능 여부 확인을 위해 Kakao 지도 서비스 또는 Kakao REST API 프록시를 통해 처리될 수 있습니다.

문의·건의 기능에 입력한 내용은 Firebase Firestore에 저장될 수 있으며, 삭제 또는 정정 요청은 개발자 연락처로 요청할 수 있습니다.

3. 처리할 수 있는 정보

구분 처리되는 정보 이용 목적 보관 여부
위치 정보 사용자가 현재 위치, 내 주변 찾기, 길찾기 기능을 사용할 때 기기 또는 브라우저가 제공하는 위치 정보 가까운 성당·성지·피정의집 표시, 거리 계산, 지도 표시, 길찾기 가톨릭길동무가 위치 정보를 별도 서버에 저장하지 않습니다.
문의 및 건의 내용 사용자가 문의 화면에 입력한 분류, 제목, 내용, 익명 작성자명, 작성일, 서버 저장 시각 문의 확인, 오류 수정, 개선 건의 검토, 앱 운영 및 품질 개선 Firebase Firestore에 저장될 수 있습니다.
앱 사용 편의 정보 즐겨찾기, 글자 크기, 나의 교구·본당 설정, 성지 순례 기록, 방문 날짜, 화면 복귀 상태, 임시 상태, 캐시 등 브라우저 저장 정보 사용자 설정 유지, 성지 순례 기록 표시, 앱 표시 속도 개선, 오프라인 또는 재접속 편의 제공 사용자의 기기 또는 브라우저에 저장될 수 있으며, 앱 서버에 별도 계정 정보로 저장하지 않습니다.
지도·길찾기 요청 장소 검색어, 출발지·도착지 좌표 또는 주소, 지도 표시와 경로 계산에 필요한 정보 장소 검색, 경로검색, 길찾기, 지도 표시 Kakao 지도 서비스 및 Kakao REST API 프록시 처리 과정에서 사용될 수 있으며, 가톨릭길동무가 별도의 위치 이력으로 보관하지 않습니다.
외부 링크 이용 성당, 성지, 피정의집, 교구, 가톨릭 웹사이트 등 외부 사이트 접속 과정에서 해당 외부 서비스가 처리하는 정보 외부 웹사이트 연결 및 관련 정보 확인 각 외부 서비스의 정책에 따릅니다.

4. 위치 정보 이용

가톨릭길동무는 사용자가 직접 내 주변 찾기, 현재 위치, 길찾기 기능을 실행할 때 위치 정보를 사용할 수 있습니다.

위치 정보는 가까운 장소 표시, 거리 계산, 출발지 설정, 지도 및 길찾기 기능 제공, 성지 순례 GPS 자동 등록 여부 확인을 위해 사용됩니다.

사용자는 기기 또는 브라우저 설정에서 위치 권한을 거부하거나 철회할 수 있습니다. 위치 권한을 거부하면 현재 위치 기반 기능 일부가 제한될 수 있습니다.

5. 문의 기능

가톨릭길동무의 문의·건의 기능은 Firebase Firestore를 이용하여 사용자가 작성한 문의, 수정 제안, 개선 건의, 칭찬 내용을 저장할 수 있습니다.

문의 기능은 익명 사용을 기본으로 하며, 앱은 별도의 회원가입이나 로그인을 요구하지 않습니다.

문의 삭제 또는 정정을 요청하려면 작성한 문의의 제목, 작성일, 분류 등 식별 가능한 정보를 함께 알려 주세요.

6. 제3자 서비스 이용

가톨릭길동무는 앱 기능 제공을 위해 다음과 같은 외부 서비스를 사용할 수 있습니다.

  • Kakao 지도 서비스: 지도 표시, 위치 검색, 길찾기 기능 제공
  • Cloudflare Workers: Kakao REST API 요청을 위한 프록시 처리
  • Firebase / Firestore: 문의 및 건의 내용 저장
  • GitHub Pages: 앱 웹페이지 제공
  • Google Fonts, Font Awesome: 글꼴 및 아이콘 표시
  • 가톨릭 관련 외부 웹사이트: 매일미사, 성가, 성경, 성지 상세정보, 교구·성당·피정의집 홈페이지 등 정보 확인을 위한 링크 연결

외부 서비스가 처리하는 정보는 해당 서비스의 개인정보처리방침 및 이용약관의 적용을 받을 수 있습니다.

7. 개인정보의 공유 및 판매

가톨릭길동무는 사용자의 개인정보를 판매하지 않습니다.

다만 법령에 따른 요청이 있거나, 앱 기능 제공을 위해 필요한 외부 서비스 이용 과정에서 관련 정보가 해당 서비스에 처리될 수 있습니다.

8. 보관 기간 및 삭제 요청

문의 및 건의 내용은 앱 운영, 오류 수정, 개선 검토에 필요한 기간 동안 보관될 수 있습니다. 성지 순례 기록, 즐겨찾기, 글자 크기, 나의 교구·본당 설정은 사용자의 기기 또는 브라우저 저장소에 보관될 수 있습니다.

사용자는 문의 내용의 삭제 또는 정정을 요청할 수 있으며, 개발자는 요청 내용을 확인한 뒤 가능한 범위에서 조치합니다.

기기 또는 브라우저에 저장된 즐겨찾기, 설정, 캐시 정보는 사용자가 브라우저 데이터 삭제, 앱 데이터 삭제, 캐시 삭제 등을 통해 삭제할 수 있습니다.

가톨릭길동무는 별도의 사용자 계정을 만들지 않으므로 계정 삭제 기능은 제공하지 않습니다.

9. 아동의 개인정보

가톨릭길동무는 아동을 주 대상으로 개인정보를 수집하지 않습니다.

아동 사용자가 문의 기능을 이용하는 경우 보호자의 지도를 권장하며, 문의 내용에 개인정보를 입력하지 않도록 안내합니다.

10. 보안

가톨릭길동무는 HTTPS 기반의 GitHub Pages를 통해 앱을 제공하며, 문의 데이터는 Firebase Firestore에 저장될 수 있습니다.

다만 인터넷을 통한 정보 전송 및 저장에는 항상 일정한 위험이 있을 수 있으므로, 문의 기능에 민감한 개인정보를 입력하지 않는 것이 좋습니다.

중요한 변경이 있는 경우 앱 또는 개인정보처리방침 페이지를 통해 안내할 수 있습니다.
